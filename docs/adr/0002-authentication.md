# ADR 0002: Authentication Approach

## Status
Accepted

## Decision

Use OSU email-based authentication to restrict marketplace access to verified students.

## Context

Milestone 1 research identified trust and scam prevention as key user concerns.

## Rationale

Restricting login to OSU emails:

- Improves safety
- Reduces scams
- Creates verified campus community
- Supports reporting and moderation features
